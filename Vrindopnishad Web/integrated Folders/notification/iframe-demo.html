Iframe Notification Demo

This page demonstrates how to send notifications from an iframe to the parent window. Click the buttons below to test different notification types.

How to Use in Your Iframe Content

Add the notification bridge script to your iframe content:

<script src="../notification/iframe-bridge.js"></script>

Then use the showIframeNotification function to show notifications:

// Basic usage
showIframeNotification('Your message here');

// With type and options
showIframeNotification('Action completed!', 'success', {
  appName: 'Your App Name',
  icon: '<i class="fas fa-check"></i>',
  duration: 5000
});
