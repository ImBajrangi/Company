Step 8: Customization

To add your own resources, just edit the project-config.js file:

// Add your CSS files RESOURCES.css.custom.myNewPage = "/css/my-new-page.css"; // Add your JS files RESOURCES.js.modules.myNewScript = "/js/my-new-script.js"; // Create new bundles BUNDLES.myNewBundle = { css: ['custom.myNewPage'], js: [{ path: 'modules.myNewScript', options: { module: true } }] };
